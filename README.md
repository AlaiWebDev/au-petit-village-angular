# Au petit village

## Description
Site e-commerce de figurines faites main inspirées de l’univers d’Astérix & Obélix.

## Objectif du devoir
Dynamiser un site web avec Angular en utilisant :
- Le routing
- Les services
- Les pipes
- Les composants
- La gestion des événements

## Technologies utilisées
- Angular (Standalone)
- TypeScript
- HTML
- CSS

## Installation

Cloner le projet :

git clone https://github.com/VOTRE-NOM/au-petit-village-angular.git

Installer les dépendances :

npm install

Lancer le serveur :

ng serve

Puis ouvrir :
http://localhost:4200
